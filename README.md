# Vrinda Store Sales Analysis Dashboard

## 📌 Overview
This project analyzes Vrinda Store sales data using Excel to uncover insights about sales performance, customer behavior, and business trends. The results are presented through an interactive dashboard to support data-driven decision-making.

## 🛠️ Tools Used
- Excel
- Pivot Tables
- Dashboard
- Data Cleaning

## ❓ Business Questions

The project aims to answer the following key questions:

- Which sales channel generates the highest revenue?
- What are the most requested product sizes?
- How does order status (Delivered, Cancelled, Refunded, Returned) impact total revenue?
- Which cities generate the highest number of orders?
- Which age group contributes the most to total sales?
- Which state has the highest number of orders?
- What are the best-selling product categories?
- How do sales and number of orders change over time?
- What is the impact of gender on total sales?

## 🔍 Project Steps

1. Data Cleaning  
Handled missing values and ensured data consistency.

2. Data Processing  
Prepared and structured the data for analysis.

3. Data Analysis using Pivot Tables  
Analyzed sales, customers, and order behavior.

4. Dashboard Creation  
Built an interactive dashboard to visualize insights.

## 📈 Key Insights

- Amazon is the top-performing sales channel in terms of revenue  
- "Set" is the best-selling product category  
- Sizes M and L are the most frequently ordered  
- Most orders are Delivered, with noticeable Cancelled and Refunded cases  
- Bengaluru generates the highest number of orders  
- Adults contribute the highest share of sales (~50%)  
- Women contribute more to total sales (64%) than men (36%)  
- There is a gradual decline in sales and orders over recent months  

## 🎯 Business Recommendations

- Focus on high-performing channels like Amazon  
- Optimize inventory for popular sizes (M & L)  
- Target female customers with marketing campaigns  
- Reduce cancellations and refunds by improving service quality  
- Investigate reasons behind the decline in sales over time
